# DressingRoll CRM

A private B2B sales workflow for UK podiatry clinics.

## Current features
- Dashboard
- START TODAY queue
- Up to 25 new clinics per working session
- Follow-up queue
- Personalised Gmail draft links
- Clinic search and filters
- Status, sample and customer tracking
- CSV export
- Browser-based persistence

## Deploy
1. Upload all files in this project to the GitHub repository root.
2. Import the repository into Vercel.
3. Vercel detects Next.js automatically.
4. Click Deploy.

## Important
This V1 stores progress in the browser using localStorage.
Export CSV regularly.
Supabase database and authentication will be added in V2.
